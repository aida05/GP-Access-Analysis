# GP Access Analysis Project Plan

## Project Goal
Analyse GP appointment access across England using NHS data to identify patterns in service demand, appointment timing, and operational performance.

## Main Questions
- Which areas have the highest appointment demand?
- Which areas have lower same-day or short-notice access?
- How does appointment mode vary across regions?
- Are there operational patterns that may explain access differences?
